# Dota 2 Win Prediction – Advanced Feature Engineering

Проект по предсказанию победы команды Radiant в матчах Dota 2 с акцентом на генерацию и сравнение признаков: традиционные vs нейросетевые.

## Задача

Бинарная классификация: предсказать, победит ли Radiant (1) или Dire (0) по данным матча.  
Метрика: ROC-AUC (и Gini = 2*ROC-AUC - 1).

## Что реализовано

- Загрузка и EDA: matches_train/test, player_df, heroes, dota_adv
- Проверка дубликатов, статистика длительности матчей
- Стратифицированное разбиение train/val
- Генерация признаков:
  - Традиционные
  - Нейросетевые
- Анализ качества признаков:
  - Корреляция с целевой переменной (radiant_win)
  - Статистика: средняя |r|, топ-признаки
  - Сравнение ROC-AUC моделей на разных наборах признаков
- Визуализация сравнения подходов
